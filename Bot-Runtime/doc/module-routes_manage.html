(inner) unless(path, middleware) → {Callback}

This middleware checks whether the requested url starts with the given path. If it does, it returns the next middleware in the chain. Otherwise, it returns the middleware that has been passed in.
Parameters:
Name Type Description
path string the path that the route starts with
middleware Callback the middleware that is to be called if the path does not match
